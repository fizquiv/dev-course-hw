Comandos usados en Git

  1. git init — Inicializa un repositorio.
  2. git add . — Agrega archivos al área de preparación.
  3. git commit -m "mensaje" — Guarda los cambios.
  4. git remote add origin URL — Conecta con GitHub.
  5. git push -u origin main — Sube cambios.

Más comandos

Comando Descripción
