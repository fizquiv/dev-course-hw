Lista de comandos vistos

  • cd — Cambiar de directorio
  • ls — Listar archivos
  • mkdir — Crear un directorio
  • touch — Crear un archivo vacío
  • rm — Eliminar archivos

Ejemplo de uso

Para crear un proyecto nuevo:

cd Documentos
mkdir mi_proyecto
cd mi_proyecto
touch index.html
